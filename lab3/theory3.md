## Описание задачи

есть сигнал - содержит множество частот, нам нужно частоты

50-150 Гц - усилить в 2 раза

350-750 - без изменений

900 - 1500 - ослабить в 2 раза

остальное - не пропускать

то есть - фильтровать

## АЧХ - амплитудно-частотная характеристика

усиление, во сколько раз фильтр изменит 

в коде freqs - частоты, gains - усиление

## ФЧХ - фазо-частотная характеристика

то, на сколько фильтр сдвигает каждую частоту по времени

линейное фчх - гарантирует, что форма сигнала сохраняется
$ϕ(ω) =-\frac{N}{2} \times ω$ - линейная, сдвигается на время N / 2. Возможно, только если коэффициенты симметричны относительно центра окна, линейный т.к. график - прямая линия

нелинейная - разные частоты сдвигаются по разному, с разной задержкой, происходит искажение сигнала

в моей задаче N = 256, значит фильтр сдвигается на 128 значений

## КИХ-фильтр - конечная импульсная характеристика

**импульсная характеристика (h)** - то, насколько предыдущие отсчёты влияют на текущее значение, конечное - конечное значение ненулевых значений

h[n] - коэффициент фильтра

$y[n]=\sum_{k=0}^{N}h[k] \times x[n−k]$

берём текущий входной отчёт x[n]ипредыдущие N отчётов x[n-k], умножаем каждый из них на соответствующие импульсные характеристики h[k], то есть коэффициент

КИХ получается симметричной так как коэффициенты h симметричны относительно центра    

firwin2 строит симметричные коэффициенты, firwin - только если один диапазон с одним услвием

также существует БИХ - бесконечная, ФЧХ обычно нелинейна, при неудачном коэффициенте нарастает до бесконечности - это неустойчивость, использует свои прошлые выходные данные, мало коэффициентов

## МНК - метод наименьших квадратов

наша АЧХ - ступенчатая форма, нужно найти коэффициенты h такие, чтобы АЧХ была как можно ближе к желаемой

идеальную ступенчатую АХЧ с мгновенными переходами конечным числом коэфов невозможно, всегда будет **ошибка**, задача её **минимизировать**

Если $H_d(w)$ - желаемая АХЧ, а реальная - $H(w)$, то ошибка - $H_d - H$

МНК - сумма квадратов ошибки, находим минимум для суммы всех ошибок, получая таким образом наиболее подходящие коэффициенты, интеграл, т.к. частота - непрерывная величина

`firls` — метод наименьших квадратов с возможностью задавать веса для разных частотных диапазонов. Ближайший аналог `firwin2`, но с большим контролем над точностью в разных полосах

Окно Хэмминга - функция плавного обрезания коэффициентов с обоих концов, сглаживает и подавляет выбросы значений (эффект Гиббса), в центре близко к 1, по краям близко к 0. Для математически точного фильтра необходимо бесконечно много коэффициентов, поэтому мы берём этот ряд и обрезаем функцией окна.

## Собственное решение

D = np.interp(freq_grid, freqs_norm / 2, gains) - достраиваем значения между известными точками

матрица косинусов - У нас есть N = 128, симметричная относительно центра, уникальных 64 + центральный коэф. для каждой из частот есть зависимость от каждого из этих уникальных коэфициентов, соответственно количество частот - строки, количество столбцов - количество уникальных коэф.





### вопросы

### Импульсная характеристика h[n]

Реакция фильтра на одиночный импульс δ[n]. Это массив коэффициентов фильтра. Полностью описывает поведение фильтра — зная h[n], знаешь всё о фильтре.

### КЧХ — комплексная частотная характеристика H(ω)

Это дискретное преобразование Фурье от импульсной характеристики:

$H(ω)=∑_{n=0}^{N}h[n]⋅e^{−iωn}$

Комплексное число для каждой частоты ω. Содержит одновременно и усиление и сдвиг фазы. В коде это результат `freqz(h)`.

**Физический смысл:** показывает как фильтр обрабатывает каждую частоту — во сколько раз меняет амплитуду и на сколько сдвигает по времени.

### АЧХ — амплитудно-частотная характеристика ∣H(ω)∣

Модуль КЧХ: $∣H(ω)∣=\sqrt{a^2+b^2}​$,где $H(ω)=a+b_i$    

В коде: `np.abs(H)`. **Физический смысл:** показывает во сколько раз фильтр усиливает или ослабляет каждую частоту. У нас: 2.0 для 50–150 Гц, 1.0 для 350–750 Гц, 0.5 для 900–1500 Гц.

### ФЧХ — фазо-частотная характеристика ∠H(ω)

Аргумент (угол) КЧХ: $∠H(ω)=arctan(\frac{b}{a}​)$

В коде: `np.angle(H)`. **Физический смысл:** показывает на сколько фильтр сдвигает каждую частоту по времени. У КИХ с симметричными коэффициентами — линейная, то есть все частоты задерживаются одинаково на N/2 отсчётов. Сигнал не искажается, только запаздывает.

### МНК — метод наименьших квадратов

Находит коэффициенты h[n] минимизируя суммарную ошибку между желаемой и реальной АЧХ: $min_h​∫_0^π​∣Hd​(ω)−H(ω)∣^2dω$

В матричном виде решается через систему A⋅a=D, где A — матрица косинусов, D — желаемая АЧХ. Решение: `np.linalg.lstsq(A, D)`.

## Описание задачи

есть сигнал - содержит множество частот, нам нужно частоты

50-150 Гц - усилить в 2 раза

350-750 - без изменений

900 - 1500 - ослабить в 2 раза

остальное - не пропускать

то есть - фильтровать

## АЧХ - амплитудно-частотная характеристика

усиление, во сколько раз фильтр изменит 

в коде freqs - частоты, gains - усиление

## ФЧХ - фазо-частотная характеристика

то, на сколько фильтр сдвигает каждую частоту по времени

линейное фчх - гарантирует, что форма сигнала сохраняется
$ϕ(ω) =-\frac{N}{2} \times ω$ - линейная, сдвигается на время N / 2. Возможно, только если коэффициенты симметричны относительно центра окна, линейный т.к. график - прямая линия

нелинейная - разные частоты сдвигаются по разному, с разной задержкой, происходит искажение сигнала

в моей задаче N = 256, значит фильтр сдвигается на 128 значений

## КИХ-фильтр - конечная импульсная характеристика

**импульсная характеристика (h)** - то, насколько предыдущие отсчёты влияют на текущее значение, конечное - конечное значение ненулевых значений

h[n] - коэффициент фильтра

$y[n]=\sum_{k=0}^{N}h[k] \times x[n−k]$

берём текущий входной отчёт x[n]ипредыдущие N отчётов x[n-k], умножаем каждый из них на соответствующие импульсные характеристики h[k], то есть коэффициент

КИХ получается симметричной так как коэффициенты h симметричны относительно центра    

firwin2 строит симметричные коэффициенты, firwin - только если один диапазон с одним услвием

также существует БИХ - бесконечная, ФЧХ обычно нелинейна, при неудачном коэффициенте нарастает до бесконечности - это неустойчивость, использует свои прошлые выходные данные, мало коэффициентов

## МНК - метод наименьших квадратов

наша АЧХ - ступенчатая форма, нужно найти коэффициенты h такие, чтобы АЧХ была как можно ближе к желаемой

идеальную ступенчатую АХЧ с мгновенными переходами конечным числом коэфов невозможно, всегда будет **ошибка**, задача её **минимизировать**

Если $H_d(w)$ - желаемая АХЧ, а реальная - $H(w)$, то ошибка - $H_d - H$

МНК - сумма квадратов ошибки, находим минимум для суммы всех ошибок, получая таким образом наиболее подходящие коэффициенты, интеграл, т.к. частота - непрерывная величина

Окно Хэмминга - функция плавного обрезания коэффициентов с обоих концов, сглаживает и подавляет выбросы значений (эффект Гиббса), в центре близко к 1, по краям близко к 0. Для математически точного фильтра необходимо бесконечно много коэффициентов, поэтому мы берём этот ряд и обрезаем функцией окна.



## Собственное решение

D = np.interp(freq_grid, freqs_norm / 2, gains) - достраиваем значения между известными точками

матрица косинусов - У нас есть N = 128, симметричная относительно центра, уникальных 64 + центральный коэф. для каждой из частот есть зависимость от каждого из этих уникальных коэфициентов, соответственно количество частот - строки, количество столбцов - количество уникальных коэф.





### вопросы

### Импульсная характеристика h[n]

Реакция фильтра на одиночный импульс δ[n]. Это массив коэффициентов фильтра. Полностью описывает поведение фильтра — зная h[n], знаешь всё о фильтре.

### КЧХ — комплексная частотная характеристика H(ω)

Это дискретное преобразование Фурье от импульсной характеристики:

$H(ω)=∑_{n=0}^{N}h[n]⋅e^{−iωn}$

Комплексное число для каждой частоты ω. Содержит одновременно и усиление и сдвиг фазы. В коде это результат `freqz(h)`.

**Физический смысл:** показывает как фильтр обрабатывает каждую частоту — во сколько раз меняет амплитуду и на сколько сдвигает по времени.

### АЧХ — амплитудно-частотная характеристика ∣H(ω)∣

Модуль КЧХ: $∣H(ω)∣=\sqrt{a^2+b^2}​$,где $H(ω)=a+b_i$    

В коде: `np.abs(H)`. **Физический смысл:** показывает во сколько раз фильтр усиливает или ослабляет каждую частоту. У нас: 2.0 для 50–150 Гц, 1.0 для 350–750 Гц, 0.5 для 900–1500 Гц.

### ФЧХ — фазо-частотная характеристика ∠H(ω)

Аргумент (угол) КЧХ: $∠H(ω)=arctan(\frac{b}{a}​)$

В коде: `np.angle(H)`. **Физический смысл:** показывает на сколько фильтр сдвигает каждую частоту по времени. У КИХ с симметричными коэффициентами — линейная, то есть все частоты задерживаются одинаково на N/2 отсчётов. Сигнал не искажается, только запаздывает.

### МНК — метод наименьших квадратов

Находит коэффициенты h[n] минимизируя суммарную ошибку между желаемой и реальной АЧХ: $min_h​∫_0^π​∣Hd​(ω)−H(ω)∣^2dω$

В матричном виде решается через систему A⋅a=D, где A — матрица косинусов, D — желаемая АЧХ. Решение: `np.linalg.lstsq(A, D)`.

import numpy as np

import matplotlib.pyplot as plt

from scipy.signal import firls, freqz, lfilter

def my_mnk_filter(N, freqs_norm, gains, num_points=4096):

"""

реализация МНК для КИХ-фильтра с линейной ФЧХ

freqs_norm - нормированные частоты [0, 1]

gains - усиления

num_points - число точек частотной сетки

"""

# число коэффициентов и центральный индекс

N_coefs = N + 1

M = N // 2 # центр симметрии

# равномерная сетка частот от 0 до 0.5 (в нормированных единицах [0, 1])

# 0.5 соответствует частоте Котельникова, выше информации нет

freq_grid = np.linspace(0, 0.5, num_points)

# желаемая АЧХ на сетке, интерполируем для каждой точки сетки

# freqs_norm идёт от 0 до 1, поэтому делим на 2 для совмещения с freq_grid

D = np.interp(freq_grid, freqs_norm / 2, gains)

# матрица косинусов A размером N_coefs * (M+1)

# для КИХ с линейной ФЧХ - косинус, синус взаимно уничтожается

# из-за симметрии коэффициентов h[k] = h[N-k]

m = np.arange(M + 1)

A = np.cos(2 * np.pi * np.outer(freq_grid, m))

# МНК - находит такое a, чтобы минимизировать сумму ошибок по всем уравнениям

a = np.linalg.lstsq(A, D, rcond=None)[0]

# превращаем a в симметричный массив коэффициентов

# центральный коэффициент — a[0]

# делим на 2, т.к. пара симм. коэфов вносит двойной вклад в АЧХ

h = np.zeros(N_coefs)

h[M] = a[0]

for m_idx in range(1, M + 1):

h[M - m_idx] = a[m_idx] / 2 # левая половина

h[M + m_idx] = a[m_idx] / 2 # правая половина

return h

Fs = 8000 # частота дискретизации входного сигнала, ГЦ

F_kot = Fs / 2 # частота котельникова, содержащая полезную информацию, выше - колебания сливаются с другими частицами

N = 128 # порядок фильтра, количество прошлых отсчетов (число коэффициентов = N+1)

freqs = np.array([

0, 49, # до диапазона 1

50, 150, # диапазон 1, 50 - 150 Гц

151, 349, # между диапазоном 1 и диапазоном 2

350, 750, # диапазон 2

751, 899, # между диапазоном 2 и диапазоном 3

900, 1500, # диапазон 3

1501, F_kot # после диапазона 3

]) / F_kot # нормализуем все частоты к диапазону [0, 1]

# множители усиления для каждого диапазона

gains = np.array([

0, 0,

2.0, 2.0,

0, 0,

1.0, 1.0,

0, 0,

0.5, 0.5,

0, 0

])

# firwin - МНК, находит такие h[n], что АЧХ фильтра максимально близка к желаемой

# получаем импульсную характеристику

h = firwin2(N + 1, freqs, gains, window='hamming')

# freqz вычисляет комплексную частотную характеристику фильтра

# на основе коэффициентов, преобразование фурье (из временной области в частотную)

# N - количество точек

w, H = freqz(h, worN=8192)

# w - нормированные от 0 до pi числа

# H - массив комплексных чисел с усилением и сдвигом

# переводим нормированные частоты обратно в Гц

freq_hz = w * F_kot / np.pi

# амплитудная характеристика, модуль комплексного H(ω), усиление фильтра

amplitude = np.abs(H)

# ручная реализация МНК

h_manual = my_mnk_filter(N, freqs, gains)

# встроенная функция firls для сравнения

h_builtin = firls(N + 1, freqs * F_kot, gains, fs=Fs)

# АЧХ обоих фильтров

w_m, H_m = freqz(h_manual, worN=8192)

w_b, H_b = freqz(h_builtin, worN=8192)

freq_hz_m = w_m * F_kot / np.pi

freq_hz_b = w_b * F_kot / np.pi

amplitude_m = np.abs(H_m)

amplitude_b = np.abs(H_b)

# создаем временную шкалу для тестового сигнала, 800 отсчетов - 0.1 секунды сигнала

t = np.linspace(0, 0.1, int(Fs * 0.1), endpoint=False)

# сам сигнал

signal = (

np.sin(2 * np.pi * 100 * t) + # диап 1

np.sin(2 * np.pi * 500 * t) + # диап 2

np.sin(2 * np.pi * 1200 * t) + # диап 3

np.sin(2 * np.pi * 2000 * t) # вне диап

)

# применяем фильтр к тестовому сигналу

# свертка сигнала с коэффициентами h

filtered_signal = lfilter(h, 1.0, signal)

# ----------------------------------------------------------------------------------------------

# визуализация

fig, axes = plt.subplots(3, 1, figsize=(12, 12))

# график 1: АЧХ собственного фильтра vs желаемая АЧХ

ax1 = axes[0]

ax1.step(freqs * F_kot, gains, 'r--', linewidth=1.5,

label='Желаемая АЧХ', where='mid')

ax1.set_xlim(0, 2000)

ax1.set_ylim(-0.1, 2.5)

ax1.set_xlabel('Частота (Гц)')

ax1.set_ylabel('Усиление')

ax1.set_title('АЧХ своего КИХ-фильтра')

ax1.legend()

ax1.grid(True, alpha=0.3)

for f in [50, 150, 350, 750, 900, 1500]:

ax1.axvline(x=f, color='gray', linestyle=':', alpha=0.5)

# график 2: сравнение своего фильтра со встроенным

ax2 = axes[1]

ax2.plot(freq_hz_m, amplitude_m, 'b-', linewidth=2,

label=f'Свой фильтр')

ax2.plot(freq_hz_b, amplitude_b, 'g--', linewidth=2,

label=f'Встроенная функция')

ax2.set_xlim(0, 2000)

ax2.set_ylim(-0.1, 2.5)

ax2.set_xlabel('Частота (Гц)')

ax2.set_ylabel('Усиление')

ax2.set_title('Сравнение своего и встроенного фильтров')

ax2.legend()

ax2.grid(True, alpha=0.3)

# график 3: тестовый сигнал до и после фильтрации

ax3 = axes[2]

ax3.plot(t[:200], signal[:200], 'b-', linewidth=1,

label='Исходный сигнал', alpha=0.7)

ax3.plot(t[:200], filtered_signal[:200], 'r-', linewidth=2,

label='Отфильтрованный сигнал')

ax3.set_xlabel('Время (сек)')

ax3.set_ylabel('Амплитуда')

ax3.set_title('Тестовый сигнал до и после фильтрации')

ax3.legend()

ax3.grid(True, alpha=0.3)

plt.tight_layout()

# результаты фильтрации

print("Результаты фильтрации")
